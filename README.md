# Java des Héros 🦸

## Description
CRM pour les Avengers permettant de gérer les civils, 
missions, incidents et super-héros.

## Prérequis
- Java 25
- MySQL / PostgreSQL
- IDE (IntelliJ, Eclipse...)

## Installation
1. Cloner le repo
2. Importer le script SQL
3. Configurer la connexion BDD (config.properties)
4. Lancer le projet

## Modules
- Civils
- Organisations
- Super-héros / Super-vilains
- Incidents
- Missions
- Rapport
- Satisfaction
- Crise / Litige
